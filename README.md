# hma-scraper-js
NodeJS script: scrapes the obfuscated proxy list at proxylist.hidemyass.com

### Inspired by
https://github.com/IDSninja/hma-scraper

### Requires
NodeJS with promise support

### Install/Usage
```
$ git clone https://github.com/olesho/hma-scraper-js
$ cd hma-scraper-js
$ npm start
```
